# Đáp án và gợi ý

## Quiz 1 — Đáp án

1: A — Quản lý trạng thái server để ứng dụng có thể kết nối.
2: A — Nếu server không chạy, ứng dụng không thể kết nối.
3: A — Stop server khi bảo trì, cập nhật hoặc xử lý lỗi.

## Quiz 2 — Đáp án

1: A — `services.msc` là công cụ quản lý services trong Windows.
2: A — `mysqladmin -u root -p shutdown` là một lệnh chuẩn để tắt MySQL (yêu cầu quyền).
3: A — `Startup/Shutdown` là mục trong MySQL Workbench dành cho thao tác này.

## Gợi ý trả lời (tự luận)

- Khi trả lời câu tự luận, nêu ví dụ cụ thể (ví dụ tên service `MySQL80`, lệnh `sudo systemctl start mysql`) và mô tả ngắn các bước kiểm tra trạng thái.
